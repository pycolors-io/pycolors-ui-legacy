## What is PyColors UI?

PyColors UI is composed of three core ideas:

1. **A unified design language**
   Color tokens, spacing, radius, typography, and semantic variables built on Tailwind CSS and OKLCH color space.

2. **A flexible component architecture**
   Reusable UI primitives based on shadcn/ui and Radix primitives (Button, Card, Input, Dialog, etc.).

3. **A documentation-first workflow**
   Powered by Fumadocs, every component and pattern can be documented, versioned, previewed, and shared.
